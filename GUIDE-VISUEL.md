## 🤖 AGRIBOT - FLUX DE CONVERSATION

```
1. Utilisateur clique sur 💬
   ↓
2. Chat s'ouvre
   ↓
3. Message bienvenue + suggestions
   ┌─────────────────────────────┐
   │ Questions rapides:          │
   │ [Produit tomates?]          │
   │ [Améliorer rendement?]      │
   │ [Agriculture urbaine?]      │
   └─────────────────────────────┘
   ↓
4. User sélectionne ou tape
   ↓
5. Message envoyé à /api/agribot
   ↓
6. OpenAI GPT-4 répond
   (ou mode démo si pas d'API key)
   ↓
7. Réponse affichée avec:
   - Recommandations produits
   - Conseils pratiques
   - Contact si besoin
```

## 🛒 FLUX DU PANIER

```
1. User clique [🛒] sur produit
   ↓
2. Vérification stock
   ↓
3. Ajout au panier (Zustand)
   ↓
4. Notification toast ✅
   ↓
5. Badge compteur mis à jour
   ↓
6. Sauvegarde localStorage
   ↓
7. User peut:
   - Continuer shopping
   - Voir panier (icône header)
   - Checkout (paiement)
```
