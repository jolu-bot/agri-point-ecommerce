# Rapport d'impact — Gabarit hebdomadaire

Usage
- Générer chaque lundi 10:00 pendant la période post‑lancement.

Structure du rapport
1. Résumé exécutif (1 paragraphe)
2. KPIs clés (7 jours):
   - Total commandes
   - Revenus totaux
   - Conversion (visites → commandes)
   - Coût SMS (si pertinent) / ROI estimé
   - Taux d'abandon de panier
3. Incidents majeurs la semaine
4. Actions prises
5. Recommandations pour la semaine suivante

Exemple — Section KPIs
| KPI | Valeur | Variation Semaine |
|-----|--------:|------------------:|
| Commandes | 1,234 | +12% |
| Revenus | 7,560,000 FCFA | +15% |

Graphiques recommandés
- Commandes par heure (ligne)
- Revenus cumulés (bar)
- SMS delivery rate (pie)

Livrable
- `impact-report-YYYY-WW.pdf` (ou MD) partagé dans Slack + dossier `reports/`

Fin.
